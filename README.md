[Migrating to SWC: A brief overview](https://blog.logrocket.com/migrating-swc-webpack-babel-overview/)

## SWC Usage

```
npm i -D @swc/core @swc/cli \
npm i -S node-fetch \
npm i -S regenerator-runtime
```

Run

```
npm start
```
